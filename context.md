# 🧠 프로젝트 기억 (context.md)

> **context.md가 뭐예요?** 지금까지 정한 것과 다음에 할 일을 적어두는 **프로젝트 메모**예요.
> **두 가지 목적:** ① 내가 이어서 하기 쉽게, ② **AI(OpenCode)에게 상황을 알려주는 자료**.
> AI는 대화가 길어지거나 **새 세션**이면 앞 내용을 잊어요. 그때 이 파일을 **먼저 붙여넣고**
> *"이걸 기준으로 이어가자"* 라고 하면 AI가 그동안의 맥락을 알고 이어서 만들어줘요.
> **작업을 멈추기 전, 아래 「다음에 할 일」만 갱신**해두면 다음이 편해요.

---

## ✅ 채운 예시 (이렇게 적으면 돼요)

**무엇을 만드나**: 사과 받기 게임 (바구니로 사과 받고 점수 올리기)

**지금까지 결정한 것**
- 조작: ← → 방향키로 바구니 이동
- 규칙: 사과 +1점 / 썩은 사과 닿으면 목숨 -1 / 목숨 3개
- Phaser 3를 CDN으로 사용, index.html 하나에 작성

**다음에 할 일**
- 목숨 다 잃으면 게임오버 화면 + 다시하기 버튼
- 10점마다 떨어지는 속도 빨라지게 (난이도)

**막혔던 것 / 메모**
- 바구니가 화면 밖으로 나감 → 좌우 경계 막기 (해결함)
- 효과음 넣기: assets/audio 폴더 소리 사용 (나중에)
