# 🎮 스틱맨 격투 게임 — 계획

> prd.md 기반, 빈 index.html에 순서대로 만들기

## 단계

- [x] **1단계: 캐릭터 선택 화면**
  - 게임 시작 시 가장 먼저 등장
  - 공격형(ATTACK) / 방어형(DEFENSE) 버튼 표시
  - P1(레드): A/D로 선택, Space로 확정
  - P2(블루): ←/→로 선택, Enter로 확정
  - 선택 완료 시 게임 시작

- [x] **2단계: 캔버스 + 스틱맨 그리기**
  - Canvas 준비, 배경 그리기 (네온 팔레트)
  - 스틱맨 2명 (머리=꽉찬 원, 팔다리=곡선)
  - 빨강(공격형) / 파랑(방어형)
  - A,D / ←,→ 이동 + 화면 경계 + 충돌 처리

- [ ] **2단계: 이동 조작**
  - P1(레드): A(좌) / D(우) / W(평타) / S(고유스킬)
  - P2(블루): ←(좌) / →(우) / ↑(평타) / ↓(고유스킬)

- [ ] **3단계: 고유 스킬 (W/키)**
  - 공격형(레드): **F키 = 앞으로 대쉬**
  - 방어형(블루): **F키 = 방어** (데미지 1/2, 스턴 면역)

- [ ] **4단계: 평타 4단 콤보**
  - 마우스 좌클릭 연속 입력 → 원 → 투 → 돌려차기 → 밀어차기
  - 각 단계별 데미지·사거리·속도 다르게
  - **평타 중 이동 불가** (공격 모션 중엔 움직일 수 없음)
  - **타격마다 앞으로 조금씩 전진** (거리 좁히기)
  - **마지막 타(밀어차기) 빗나가면 1초 스턴** (헛쳤을 때 페널티)

- [ ] **5단계: HP + 데미지 판정**
  - 공격 판정 (팔/다리가 상대에게 닿았는지)
  - HP 바 그리기 (상단)
  - **평타 맞으면 0.8초 스턴** (행동 불가)
  - **맞은 쪽도 같은 거리만큼 밀려남** (때린 쪽과 동일하게 밀림)

- [ ] **6단계: 라운드 시스템**
  - 1라운드당 90초 제한
  - HP 0 = KO, 시간초과 = HP 많은 쪽 승리
  - 3판 2선승제
  - 승리/게임오버 화면

- [ ] **7단계: 마무리 (사운드 + 효과)**
  - 타격 효과음 (assets/audio/)
  - 화면 흔들림 (히트 시)
  - localStorage 최고점수?(대전 게임이니 승리 기록)
